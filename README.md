# 🎨 PromptAI Frontend | Discovery Engine & Identity Dashboard

A premium, cinematic interface for discovering, sharing, and curating the world’s most advanced AI prompts. Built with a mobile-first philosophy and a focus on high-fidelity user experiences.

## 🛠️ Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS with a bespoke glassmorphic design system
- **State**: React Context API (Auth & Global UI states)
- **Icons**: Lucide React
- **Animations**: CSS-driven micro-interactions

## 📦 Key Features
- **Root Discovery Feed**: Unified home page providing immediate access to curated prompts for both guests and members.
- **Identity Dashboard 2.0**: Premium, tabbed profile interface for managing bios, avatars, and private curation vaults.
- **Infinite Discovery**: Smooth, high-performance feed loading with real-time category filtering and global search.
- **Molecular Interactivity**: One-click prompt copying, hearting (Likes), and persistent collections (Saves).
- **Cinematic UX**: Dark-mode gradient aesthetics, glassmorphic card layouts, and mobile-optimized navigation.

## 🚀 Getting Started

### 1. Prerequisites
- Node.js (v18+)
- Backend API running at `http://localhost:5001`

### 2. Installation
```bash
npm install
```

### 3. Environment Setup
Create a `.env.local` file in this directory:
```env
NEXT_PUBLIC_API_URL=http://localhost:5001/api
```

### 4. Start Development
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the Discovery Engine.
